This project is API-first. Static assets are not required; this placeholder prevents Vercel from failing with
"No Output Directory named 'public'" when a static output directory is configured.

Health check endpoint: /health (or /api/health depending on rewrites).

If you are seeing this in production and don't need a static site, remove the Output Directory setting in the Vercel Project → Settings → Build & Development Settings.
